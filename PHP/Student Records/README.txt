Student Records - Database and Web Interface

Description:
A system for storing student records in a database. The database is used to populate a list of students, as well as retrieve academic information for them. This project was done as a learning exercise using PHP and MySQL on a Linux environment.

- The main file to run is « selectStudent.php ». 
- Records are displayed by POST to « displayRecords.php ». 
- All database related functions are in the file « dbFunctions.php ».
- A database dump is included « studentdb.sql »
